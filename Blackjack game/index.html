Blackjack Rules

The BlackJack card game as you know it, isn't rocket science to comprehend. It follows a very simple

set of rules which are laid down for y'all to go through:



  1. Each number card holds a respective value which equals the number, like, value of card 2=2, value of card 3=3 etc
  2. All the face cards hold a standard value of 10
  3. The user begins laying down cards which are randomly generated(Ace to King)
  4. The values of the cards keep adding up to the user score.
  5. The score of the user in one attempt should be less than or equal to 21. If it crosses 21, then the user is busted
  6. The user can begin betting the cards as long as he is safe about the 21 score limit. When they are done, they can hit the hold button
  7. Next it's the bot's turn. If the score of Bot exceed that of user, user loses and the vice versa
  8. If both bot and user are busted, then it's a draw. If either one of em remain busted, then the busted player loses the round
  9. The scores of user are recorded below. Press deal to start the next hand
